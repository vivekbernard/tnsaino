Do's & Don'ts

Usage guidelines to maintain design coherence and brand integrity.

Do
Use Trust Blue (#1B5FBF) for primary CTAs only — never dilute it with decorative use.
Use neutral-heavy layouts; let blue accents surface what matters most.
Maintain 8px radius on cards — not rounder, not sharper.
Apply 150–180ms transitions for all interactive states.
Use amber only for urgency: "Urgent", "New", alerts, job deadlines.
Keep typography tight and hierarchy sharp. Bold for headings, 400 for body.
Use skeletons for async content — never spinners for full-page loads.
Ensure all interactive elements have visible focus rings for keyboard users.
Don't
Don't use pill-shaped buttons (border-radius > 10px) — they read as casual, not enterprise.
Don't use blue for decorative borders, dividers, or non-interactive elements.
Don't use amber as a primary brand color or overuse it — it must feel exceptional.
Don't use bounce or spring animations — use ease-out cubic bezier only.
Don't use more than 3 font weights in a single component.
Don't use gradient buttons — flat fills with subtle shadow only.
Don't place text on amber backgrounds below a 4.5:1 contrast ratio.
Don't rely on color alone to convey status — always pair with icons or labels.

Motion Principles

Duration
Fast: 120ms — Hover, focus rings
Normal: 180ms — Buttons, cards
Slow: 250–300ms — Panel reveals
Easing
Always ease-out (decelerate).
cubic-bezier(0.16, 1, 0.3, 1)
Never bounce. Never spring.
Properties
Animate: opacity, transform, box-shadow, background-color, border-color.
Never animate: width, height, padding.
Toggle dark mode using the switch in the top-right corner to preview all components in dark theme.
